## Adding More Music Files

Place ambient music files (MP3, WAV, or OGG format) in this directory and update the `availableTracks` array in `/frontend/src/assets/js/ambientMusic.js`.

### File Requirements

- **Format**: MP3, WAV, or OGG
- **Size**: Keep files under 10MB for optimal loading
- **Length**: 5-10 minutes (files will loop automatically)
- **Volume**: Pre-normalize to comfortable listening levels

## Usage

The ambient music player is automatically enabled for the Kairos profile and can be controlled from the chat interface with play/pause controls.
